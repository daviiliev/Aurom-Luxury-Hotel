# 🏨 AURUM – Luxury Hotel Landing Page

A premium, fully responsive landing page for a luxury hotel. Built with **Vanilla HTML, CSS & JavaScript**.

![Desktop View](screenshots/desktop.png)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏨 **Fullscreen Hero** | Immersive hero section with centered content |
| 📜 **Timeline** | Visual journey of the hotel experience |
| ❓ **FAQ Accordion** | Interactive room details with smooth transitions |
| 🔄 **360° View** | Interactive panorama preview |
| 📅 **Booking Form** | Functional reservation form |
| ✨ **Scroll-Reveal** | Smooth animations on scroll |
| 📱 **Fully Responsive** | Perfect on all devices |
| 🎨 **Luxury Design** | Gold, cream & black palette |
